> Youtube transcript API based on youtube-transcript

### 🏠 [Homepage](https://github.com/ngocsangyem/transcript-api)

### ✨ [Demo](https://github.com/ngocsangyem/transcript-api)

## Usage

```sh
/api/transcript/{your_youtube_id}
```

Return type

```typescript
type Transcript = {
  start_ms: number;
  end_ms: number;
  text: string;
}
```

## Install

```sh
npm install
```

## Run tests

```sh
npm run test
```
